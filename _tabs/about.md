## 简历
我是(`morric`)一名全栈开发者，现在专注于后端架构 、 移动APP、计算机视觉 方向。

作为一个全方位技术背景的全栈开发者，我在后端开发领域深耕细作，精通JAVA、PHP和Go语言。利用JAVA的健壮性，我开发了多个大规模、高并发的系统服务。同时，借助PHP的灵活性与Go的高效性，我实现了各类Web服务和微服务架构，以高性能、高可用性满足了业务需求。

前端方面，我对Web前端技术有着丰富的经验，包括熟练使用JavaScript、HTML5和CSS3等技术，以创建响应快捷且用户友好的网站界面。在移动端，我利用Android和iOS平台的原生开发能力，构建了多款流畅并且功能丰富的应用程序。

我的技术实力不仅仅局限于单一领域，我能够跨界融合各种技术栈，无论是互联网应用、移动App还是底层服务，都能够自如应对。在追求技术卓越的同时，我也不忘关注产品的商业价值和用户体验，力求在满足用户需求的同时，也实现技术上的突破和创新。

长期企业管理类项目经验，现在专注于 JAVA 和 移动原生 开发并将长期发展下去。

### 工作经历
技术的淘汰
: 我接触编程的时候是高中学开始的，当时流行的是Flash的各种动画，要做好动画必须掌握一些编程的基础。现在还记得当时把的自己的照片做成相册、需要加鼠标动画效果，最后完成的激动的心情。

: 大学里我参加Flash的比赛，从那时奠定了我对编程基础，大学的时候开始学C#，当时用来开发网站asp.net,因为我的专业不是计算机专业，当时我做了一个计算兴趣小组的网站。从那时起我就爱上了计算机。

: 参加工作我第一份编程工作是做儿童内容，开发的语言也是Flash，当时的Flash 游戏很火。儿童平板除了学习内容，里面的游戏和交互编程的工作很多。从那时起真正的算上开发的工作。当时公司融资的失败，市场不好。写了2年的FLash，我决定离职。

: 苹果宣布不用FLash， Adobe 也宣布放弃。一门语言的彻底终结，当时我也很迷茫，迷茫一段时间后决定还是走编程的道路，开始学习IOS的开发，当时PHP比较流行，后端就学习PHP。大概自学两个多月，写了一些demo后我开始找工作，当时IOS 开发人员少我很快就找到工作。这段工作的人员变动，人员少，随着人员的离职，把Android 和 asp 的工作都交给我。当时为了工作，我开始学习Android，在学习的过程中我发现里面的设计模式和IOS的UI模式都一样。利用晚上的时间很快就把Android学完。并接手公司的项目进行维护。asp 是大学学习的，所以维护起来没有什么困难。

: 开始创业，看到一个大环境的机遇，互联网在快速发展的时期，选择创业，当时公司做网站、APP、公众号业务，为了快速进行业务发展，公众号选择一个开源的PHP平台进行二次开发，这次创业也进行技术一个全栈的发展，随着客户的要求有的要JAVA开发。为了公司发展，有开始学习JAVA，因为有。net和数据库的基础，学起来很快就掌握。创业两年由于对市场不理解，创业团队的各种问题，决定关闭公司。

: 后面找的工作都是技术管理岗位，不但要知道前缘的技术的学习还要在架构上的学习，到目前JAVA 从单体架 SOA架构 、 微服务 等一系列的学习。后端期间又学习了Golang 语言。随着业务的发展，又学习大数据，Hadoop Hive Fink 。人工智能的发展学习人工智能的方面。原生开发语言的变化开始学习kotin、swfit不断的在学习的道路。在管理岗位，不断的突破现有的技术。也是我这几年不断发展的变化。
